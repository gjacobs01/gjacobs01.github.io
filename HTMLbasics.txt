//---------------basics and key concepts-----------

all elements of a webpage are just pieces of content wrappened in flags
for example, <p> helloworld! </p> , is an example of a simple sentance wrapped
in the <p> and </p> flag, which denotes start of and end of a paragraph

Some elements do NOT have a closing flag. Images for example use just <img>.

//-------------what should happen on every new website and html page-----------------------

Set up workspace:
    in project folder/repo, make a file name index.html

    - .html denotes the file type
    - index denotes the MAIN PAGE for the website (important)

Declare document type:
    doctype dictates what html version browsers should render page using
    older html versions use verbose tags, however, all html 5 uses is:

    <!DOCTYPE html>

Create root <html> element:
    <html> and <\html> are the root elements for the entire page.
    Ideally, use <html lang = "en"> over <html> to specify language for accesebility.

Create meta <head> element:
    Remember, meta + information = information that describes information, hence meta,
    
    <html> <head> meta tags <\head> <\html>

    always include these in the meta tags area.
    <meta charset="UTF-8"> + <title>webpage tile</title>
